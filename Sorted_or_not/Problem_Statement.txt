Given an array of "n" integers, print "1" if its sorted and "0" if its not sorted.

Input Format

single line of space separated integers.

1st integer is "n" next "n" integers are elements of the array

Constraints

1<=n<=100

Output Format

a single integer, 0 or 1

Sample Input 0

3 1 2 3
Sample Output 0

1
Explanation 0

Array is sorted. So, print "1".

Sample Input 1

4 1 2 3 1
Sample Output 1

0
Explanation 1

Array is not sorted. So, print "0".
